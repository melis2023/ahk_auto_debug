# ahk_auto_debug
ahk_auto_debug for opencode 、cloud code、openclaw

这个工具是使用在opencode等编程工具中，实现自动化修复的一个工具
在ahkv1，v2版本在自动化修复中，会出现msgbox等弹窗导致进程阻塞，利用该技能可以自动识别弹窗信息，并返回给大模型进行分析并修复。

This tool is used in programming tools such as OpenCode to implement automated repair.
In AutoHotkey V1 and V2, pop‑up windows such as MsgBox may cause process blocking during automated repair.
This skill can automatically identify pop‑up window information, return it to the large language model for analysis, and then perform fixes accordingly.
