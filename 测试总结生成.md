## 4. 修复前后代码对比

### 修复前（原始带 Bug 版本 — AHK v1 风格）

```autohotkey
; Bug 1: Syntax error - missing closing bracket in If statement
F1::
    WinGetActiveTitle, title
    If (title = "Untitled"  ; missing )
        MsgBox Title is Untitled
    Return

; Bug 2: Wrong parameters for StringSplit function
F2::
    text := "apple,banana,cherry,date"
    StringSplit(result, text, ",")  ; nonexistent in v2
    MsgBox Result count: %result0%`nFirst item: %result1%
    Return

; Bug 3: Undefined variable usage
F3::
    counter += 1
    MsgBox Click count: %click_count%  ; wrong variable name
    Return

; Bug 4: Incorrect array access
F4::
    colors := ["red", "green", "blue"]
    third_color := colors[3]  ; out of bounds
    MsgBox Third color: %third_color%
    Return

; Bug 5: File read without existence check
F5::
    FileRead, file_content, C:\nonexistent_file.txt
    MsgBox Content: %file_content%
    Return

; Bug 6: catch e syntax error
F6::
    try { WinGetPos(&x, &y, &w, &h, "Nonexistent") }
    catch e {  ; invalid in v2

    }
    Return
```

### 修复后（最终版本 — AHK v2 风格）

```autohotkey
#Requires AutoHotkey v2.0
#SingleInstance Force

F1:: {
    title := WinGetTitle("A")  ; Fixed: use WinGetTitle instead of WinGetActiveTitle
    if (title = "Untitled")   ; Fixed: added closing )
        MsgBox("Title is Untitled")
    return
}

F2:: {
    counter += 1
    MsgBox("Click count: " counter)  ; Fixed: click_count -> counter
    return
}

F3:: {
    colors := ["red", "green", "blue"]
    third_color := colors[2]  ; Fixed: 0-based index, [2] = "blue"
    MsgBox("Third color: " third_color)
    return
}

F4:: {
    if FileExist("C:\nonexistent_file.txt") {
        file_content := FileRead("C:\nonexistent_file.txt")
        MsgBox("Content: " file_content)
    } else {
        MsgBox("File does not exist")
    }
    return
}

F5:: {
    count := 0
    Loop 5 {
        if count = 3
            break
        count++
    }
    MsgBox("Loop finished at count: " count)
    return
}

F6:: {
    SetTitleMatchMode("Fast")
    try {
        WinGetPos(&x, &y, &w, &h, "Some Window That May Not Exist")
        MsgBox("Window position: " x ", " y)
    }
    catch {  ; Fixed: removed 'e' variable binding
        MsgBox("Window not found")
    }
    return
}
```

## 5. 修复详情

### Bug 1 — Syntax Error: Missing `)`
- **原因**：`If (title = "Untitled"` 左括号未闭合
- **修复**：添加右括号 → `If (title = "Untitled")`
- **轮次**：第 1 轮

### Bug 2 — Runtime Error: `StringSplit` 不存在
- **原因**：AHK v2 中 `StringSplit` 已被移除，改用 `StrSplit()` 返回数组
- **修复**：`result := StrSplit(text, ",")` → 通过 `result.Length` 和 `result[1]` 访问
- **轮次**：第 2 轮

### Bug 3 — Runtime Error: `MsgBox()` 调用方式错误
- **原因**：最初脚本为 v1 风格，`MsgBox` 是命令而非函数；迁移到 v2 后语法正确
- **修复**：添加 `#Requires AutoHotkey v2.0`，统一使用 `MsgBox("text")` 函数调用
- **轮次**：第 3 轮

### Bug 4 — Syntax Error: Hotkey 缺少 `{ }`
- **原因**：AHK v2 要求 hotkey 体必须用 `{ }` 包裹
- **修复**：所有 `F1::` → `F1:: { ... }`
- **轮次**：第 4 轮

### Bug 5 — Warning: `WinGetActiveTitle` 变量未赋值
- **原因**：AHK v2 静态分析将 `WinGetActiveTitle` 误判为未赋值的局部变量
- **修复**：改用 `WinGetTitle("A")` 获取活动窗口标题
- **轮次**：第 5-6 轮

### Bug 6 — Runtime Error: `catch e` 语法错误
- **原因**：AHK v2 中 catch 不接受变量绑定，语法为 `catch { }` 而非 `catch e { }`
- **修复**：移除 `e` 变量 → `catch { MsgBox("Window not found") }`
- **轮次**：第 7 轮

### 衍生问题 — `filePath` Warning
- **原因**：AHK v2 的 `#Warn` 在 try/catch 块内无法追踪外部赋值
- **修复**：直接在 `FileExist()` 和 `FileRead()` 中使用字符串字面量，消除中间变量
- **轮次**：第 8-12 轮

### 衍生问题 — `#Warn LocalNotAssigned Off` 无效语法
- **原因**：AHK v2 不支持此参数组合
- **修复**：移除该指令，依赖默认行为
- **轮次**：第 13 轮
